**Qubit MABK Inequality Violation (Quantum Computing)**

*THIS PROJECT WAS DONE FOR MY RESEARCH INTERNSHIP AT THE CAMBRIDGE CENTRE OF INTERNATIONAL RESEARCH*

This project demonstrates the violation of the Mermin–Ardehali–Belinskii–Klyshko (MABK) inequality using multi-qubit quantum circuits. It simulates entangled states and evaluates expectation values that exceed classical bounds, highlighting non-local correlations in quantum mechanics.

🧠 What this shows
Construction of multi-qubit GHZ states
Evaluation of MABK inequality operators
Comparison between classical vs quantum bounds
Numerical simulation using quantum circuits

⚛️ Key idea
Classical local hidden variable theories impose a strict upper bound on correlations. This project shows how entangled qubit systems violate that bound, reinforcing the non-classical nature of quantum mechanics.

🛠️ Tech
Quantum circuit simulation using Qiskit
Linear algebra + expectation value computation

📈 Result
The computed MABK value exceeds the classical limit, demonstrating a clear violation consistent with quantum predictions.
